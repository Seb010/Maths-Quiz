# 🧮 Quadratic Formula Quiz Application

A desktop mathematics quiz application built with **Python** using **Tkinter** for the graphical user interface.

The application tests users’ understanding of the quadratic formula and tracks performance using a leaderboard system backed by a database.

---

## 🚀 Technologies Used

- Python
- Tkinter (GUI)
- SQLite (or your database system)
- Object-Oriented Programming

---

## 📌 Features

### Quiz System
- Randomly generated quadratic equation questions
- User input validation
- Immediate answer feedback
- Score calculation

### Leaderboard
- Persistent score storage in database
- Ranking system based on performance
- Display of top-performing users

### User Interface
- Clean and simple Tkinter GUI
- Interactive buttons and input fields
- Structured quiz flow
